**Broken**

This implemtation only supports integers, if there are any numbers after the decomal point they will be rounded down

#### Special Buttons
* EXIT - Closes Program
* RCL - AC
* S<=>D - C
* ( - POP
* ) - SWAP
* . - SQRT
* *10^x - POW 
* (-) - Percent
